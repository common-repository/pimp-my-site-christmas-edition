== Installation ==
= AUTOMATIC INSTALLATION =
Automatic installation is the easiest option — WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Pimp my Site - Christmas Edition, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Pimp my Site - Christmas Edition” then click “Search Plugins.” Once you’ve found us, you can install it by clicking “Install Now,” and WordPress will take it from there.

= MANUAL INSTALLATION =
Manual installation method requires downloading the Pimp my Site - Christmas Edition plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation-1).

= UPDATING =
Automatic updates should work smoothly, but we still recommend you back up your site.

== Frequently Asked Questions ==
= How do I choose a preset? =
+ From the WordPress Dashboard, go to "Settings > Pimp my Site".
+ Click "Choose your preset".
+ Click on your preferred preset from the list and confirm to apply it.
= Where are the settings? =
+ You can access the settings from the WordPress Dashboard, via "Settings > Pimp my Site".
